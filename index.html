支持的开发工具

KAT-Coder模型支持与主流开发工具集成

Claude Code

通过环境变量配置使用KAT-Coder模型

  • 支持macOS和Linux
  • 命令行工具
  • 环境变量配置

Cline

VS Code插件，支持自定义API Key

  • VS Code扩展
  • 图形界面配置
  • 实时代码生成

Kilo Code

IDE插件，支持多种开发环境

  • 多IDE支持
  • 自定义配置
  • 代码辅助功能

Roo Code

扩展插件，支持权限配置

  • 权限管理
  • API配置
  • 多种编程任务
