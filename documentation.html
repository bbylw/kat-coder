开始使用 KAT-Coder

KAT-Coder 系列模型专为 Agentic Coding 任务设计。KAT-Coder-Pro 和 KAT-Coder-Air 模型支持与 Claude Code、Cline、Kilo Code 和 Roo Code 集成。

从 Vanchin 平台获取 API Key 并创建推理端点后，您就可以在各种 Agent 产品中配置和使用。

注意：详细的配置步骤请参考集成指南页面。

配置指南

按照以下步骤配置 KAT-Coder 模型与开发工具的集成：

步骤一：获取 Vanchin API Key

  1. 登录 Vanchin 平台
  2. 点击左侧菜单 [API Key]
  3. 点击"Create API Key"
  4. 安全保管生成的 API Key

步骤二：创建推理端点

  1. 登录 Vanchin 平台
  2. 点击左侧菜单 [模型] → [推理]
  3. 点击"创建推理端点"
  4. 选择 KAT-Coder 模型
  5. 配置推理参数
  6. 复制推理端点 ID

步骤三：配置开发工具

根据您使用的开发工具，参考集成指南进行详细配置。

环境变量配置示例： 
export VC_API_KEY="your-api-key-here"

Vanchin 平台 API 配置

KAT-Coder 系列模型通过 Vanchin 平台提供服务。以下是详细的 API 配置信息：

获取 API Key

登录 Vanchin 平台 > 左侧菜单 [API Key] > 点击"Create API Key"

字段 描述
名称 系统生成，不可编辑
权限 目前仅支持全部（可访问项目下的所有资源）
重要：请安全保管生成的 API Key

环境变量配置（推荐）

export VC_API_KEY="your-api-key-here"

准备推理端点

登录 Vanchin 平台 > 左侧菜单 [模型] → [推理] > 点击"创建推理端点"

  1. 选择 KAT-Coder 模型
  2. 配置推理参数
  3. 复制推理端点 ID

API 调用示例

方法1：使用 curl 命令

curl 'https://vanchin.streamlake.ai/api/gateway/v1/endpoints/chat/completions' \
-H "Authorization: Bearer $VC_API_KEY" \
-H 'Content-Type: application/json' \
-d '{
    "model": "ep-59dwym-1761224975357840539", 
    "messages": [ 
        {
            "role": "system",
            "content": "You are an AI assistant."
        },
        {
            "role": "user",
            "content": "Please introduce the eight planets of the solar system"
        }
    ]
}'

方法2：使用 OpenAI Python SDK

import os
from openai import OpenAI

# 初始化 OpenAI 客户端
client = OpenAI(
    base_url="https://vanchin.streamlake.ai/api/gateway/v1/endpoints",
    api_key=os.environ.get("VC_API_KEY")
)

# 单轮对话
completion = client.chat.completions.create(
    model="ep-59dwym-1761224975357840539",
    messages=[
        {"role": "system", "content": "You are an AI assistant"},
        {"role": "user", "content": "Please introduce the eight planets of the solar system"},
    ],
)
print(completion.choices[0].message.content)
参数说明：
  • model: 推理端点 ID（从 Vanchin 平台获取）
  • messages: 对话消息数组，包含 system 和 user 角色
Python SDK 要求：
  • 安装：pip install openai
  • 设置环境变量：export VC_API_KEY="your-api-key-here"
  • 确保网络连接正常，可以访问 Vanchin 平台 API
