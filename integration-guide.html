KAT-Coder 开发工具集成指南

KAT-Coder 系列模型专为 Agentic Coding 任务设计。其中 KAT-Coder-Pro 和 KAT-Coder-Air 模型支持与 Claude Code、Cline、Kilo Code 和 Roo Code 集成。

从 Vanchin 平台获取 API Key 并创建推理端点后，您就可以在各种 Agent 产品中配置和使用。

获取 Vanchin API Key 并创建 KAT-Coder 推理端点

一、获取和配置 API Key

获取 API Key

登录 Vanchin 平台 > 左侧菜单 [API Key] > 点击"Create API Key"

字段 描述
名称 系统生成，不可编辑
权限 目前仅支持全部（可访问项目下的所有资源）
重要：请安全保管生成的 API Key。

配置 API Key

环境变量配置（推荐）：
export VC_API_KEY="your-api-key-here"

Python SDK 调用示例

使用 OpenAI Python SDK：
import os
from openai import OpenAI

# 初始化 OpenAI 客户端
client = OpenAI(
    base_url="https://vanchin.streamlake.ai/api/gateway/v1/endpoints",
    api_key=os.environ.get("VC_API_KEY")
)

# 单轮对话
completion = client.chat.completions.create(
    model="ep-59dwym-1761224975357840539",
    messages=[
        {"role": "system", "content": "You are an AI assistant"},
        {"role": "user", "content": "Please introduce the eight planets of the solar system"},
    ],
)
print(completion.choices[0].message.content)
安装命令：pip install openai

二、准备推理端点

准备在线推理端点

登录 Vanchin 平台 > 左侧菜单 [模型] → [推理] > 点击"创建推理端点"

选择 KAT-Coder 模型。

在在线推理页面，找到相应的推理端点并复制推理端点 ID。

Claude Code

配置以下环境变量以在 Claude Code 中使用 Vanchin 提供的 KAT-Coder 模型：

  • ANTHROPIC_BASE_URL: 设置为您的 Vanchin 推理端点 URL
  • ANTHROPIC_AUTH_TOKEN: 设置为您的 Vanchin API KEY
  • ANTHROPIC_MODEL & ANTHROPIC_SMALL_FAST_MODEL: 设置为 "KAT-Coder"
注意：目前仅支持 KAT-Coder 模型。您无法通过 ANTHROPIC_MODEL 环境变量修改使用的模型。

一、Claude Code 安装和配置

macOS Windows
npm install -g @anthropic-ai/claude-code Claude Code only supports macOS and Linux by default. You need to install WSL or Git for Windows first. Please refer to the official documentation.
export ANTHROPIC_BASE_URL="https://vanchin.streamlake.ai/api/gateway/v1/endpoints/ep-59dwym-1761224975357840539/claude-code-proxy"
使用您的 Vanchin 推理端点 ID
export ANTHROPIC_AUTH_TOKEN="$VC_API_KEY" setx ANTHROPIC_AUTH_TOKEN "$VC_API_KEY"
集成文档：详细集成文档请参考：https://www.streamlake.ai/document/DOC/mg6k6nlp8j6qxicx4c9

二、运行 Claude Code

在终端中运行以下命令启动 Claude Code：

claude
运行前检查 shell 类型并确保所有环境变量都已正确设置。
