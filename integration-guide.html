KAT-Coder 开发工具集成指南

KAT-Coder 系列模型专为 Agentic Coding 任务设计。其中 KAT-Coder-Pro 和 KAT-Coder-Air 模型支持与 Claude Code、Cline、Kilo Code 和 Roo Code 集成。

从 Vanchin 平台获取 API Key 并创建推理端点后，您就可以在各种 Agent 产品中配置和使用。

获取 Vanchin API Key 并创建 KAT-Coder 推理端点

一、获取和配置 API Key

获取 API Key

登录 Vanchin 平台 > 左侧菜单 [API Key] > 点击"Create API Key"

字段 描述
名称 系统生成，不可编辑
权限 目前仅支持全部（可访问项目下的所有资源）
重要：请安全保管生成的 API Key。

配置 API Key

环境变量配置（推荐）：
export VC_API_KEY="your-api-key-here"
提示：将此环境变量添加到您的 shell 配置文件中（如 ~/.bashrc、~/.zshrc 或 ~/.bash_profile）

Python SDK 调用示例

使用 OpenAI Python SDK：
安装命令 pip install openai 
import os
from openai import OpenAI

# 初始化 OpenAI 客户端
client = OpenAI(
    base_url="https://vanchin.streamlake.ai/api/gateway/v1/endpoints",
    api_key=os.environ.get("VC_API_KEY")
)

# 单轮对话
completion = client.chat.completions.create(
    model="ep-59dwym-1761224975357840539",
    messages=[
        {"role": "system", "content": "You are an AI assistant"},
        {"role": "user", "content": "Please introduce the eight planets of the solar system"},
    ],
)
print(completion.choices[0].message.content)
注意：请将 ep-59dwym-1761224975357840539 替换为您自己的推理端点 ID

流式输出示例

Python SDK 流式调用：
import os
from openai import OpenAI

# 初始化 OpenAI 客户端
client = OpenAI(
    base_url="https://vanchin.streamlake.ai/api/gateway/v1/endpoints",
    api_key=os.environ.get("VC_API_KEY")
)

# 流式对话
completion = client.chat.completions.create(
    model="ep-59dwym-1761224975357840539",
    messages=[
        {"role": "system", "content": "You are an AI assistant"},
        {"role": "user", "content": "Please introduce the eight planets of the solar system"},
    ],
    stream=True  # 启用流式输出
)

for chunk in completion:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="", flush=True)
print()

REST API 调用示例

使用 curl 命令：
curl -X POST "https://vanchin.streamlake.ai/api/gateway/v1/endpoints" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $VC_API_KEY" \
  -d '{
    "model": "ep-59dwym-1761224975357840539",
    "messages": [
      {"role": "system", "content": "You are an AI assistant"},
      {"role": "user", "content": "Please introduce the eight planets of the solar system"}
    ],
    "stream": false
  }'
将 stream 设置为 true 可启用流式输出

二、准备推理端点

准备在线推理端点

登录 Vanchin 平台 > 左侧菜单 [模型] → [推理] > 点击"创建推理端点"

选择 KAT-Coder 模型。

在在线推理页面，找到相应的推理端点并复制推理端点 ID。

Claude Code

在 Claude Code 中使用 Vanchin 提供的 KAT-Coder 模型，请配置以下环境变量：

● ANTHROPIC_BASE_URL: 设置为 https://vanchin.streamlake.ai/api/gateway/v1/endpoints/ep-xxx-xxx/claude-code-proxy，将 ep-xxx-xxx 替换为您的 Vanchin 推理端点 ID
● ANTHROPIC_AUTH_TOKEN: 设置为您的 Vanchin API KEY
● ANTHROPIC_MODEL & ANTHROPIC_SMALL_FAST_MODEL: 设置为 "KAT-Coder"
注意：目前仅支持 KAT-Coder 模型。您无法通过 ANTHROPIC_MODEL 环境变量修改使用的模型。

macOS

1. Claude Code 安装
npm install -g @anthropic-ai/claude-code
2. 配置环境变量

将 ep-xxx-xxx 替换为您的 Vanchin 推理端点 ID，并将 YOUR_VANCHIN_API_KEY 替换为您的 Vanchin API 密钥

# Check shell type
echo $SHELL

# For Zsh
echo 'export ANTHROPIC_BASE_URL="https://vanchin.streamlake.ai/api/gateway/v1/endpoints/ep-xxx-xxx/claude-code-proxy"' >> ~/.zshrc
echo 'export ANTHROPIC_AUTH_TOKEN="YOUR_VANCHIN_API_KEY"' >> ~/.zshrc
echo 'export ANTHROPIC_MODEL="KAT-Coder"' >> ~/.zshrc
echo 'export ANTHROPIC_SMALL_FAST_MODEL="KAT-Coder"' >> ~/.zshrc
source ~/.zshrc

# For Bash
echo 'export ANTHROPIC_BASE_URL="https://vanchin.streamlake.ai/api/gateway/v1/endpoints/ep-xxx-xxx/claude-code-proxy"' >> ~/.bash_profile
echo "export ANTHROPIC_AUTH_TOKEN='YOUR_VANCHIN_API_KEY'" >> ~/.bash_profile
echo 'export ANTHROPIC_MODEL="KAT-Coder"' >> ~/.bash_profile
echo 'export ANTHROPIC_SMALL_FAST_MODEL="KAT-Coder"' >> ~/.bash_profile
source ~/.bash_profile

Windows

1. Claude Code 安装

Claude Code 默认仅支持 macOS 和 Linux。您需要先安装 WSL 或 Git for Windows。请参阅官方文档

2. 配置环境变量

将 ep-xxx-xxx 替换为您的 Vanchin 推理端点 ID，并将 YOUR_VANCHIN_API_KEY 替换为您的 Vanchin API 密钥

setx ANTHROPIC_AUTH_TOKEN "YOUR_VANCHIN_API_KEY"
setx ANTHROPIC_BASE_URL "https://vanchin.streamlake.ai/api/gateway/v1/endpoints/ep-xxx-xxx/claude-code-proxy"
setx ANTHROPIC_MODEL "KAT-Coder"
setx ANTHROPIC_SMALL_FAST_MODEL "KAT-Coder"

II. 运行 Claude Code

在终端中运行以下命令以启动 Claude Code：

claude

Cline

Cline 是 VS Code 的 AI 编程助手扩展，可与 KAT-Coder 模型无缝集成。

一、安装 Cline 插件

1
安装 Cline 扩展

在 VS Code 中搜索并安装 Cline 扩展：

1. 打开 VS Code
2. 按 Ctrl+Shift+X 打开扩展面板
3. 搜索 "Cline"
4. 点击 "安装" 按钮

二、配置 API 设置

1
选择 API Key 方法

选择"Use your own API Key"

2
配置 API Key

在 VS Code 设置中添加以下配置：

{
  "cline.apiKey": "your-vanchin-api-key",
  "cline.apiBaseUrl": "https://vanchin.streamlake.ai/api/gateway/v1/endpoints",
  "cline.model": "ep-59dwym-1761224975357840539"
}
请将 ep-59dwym-1761224975357840539 替换为您自己的推理端点 ID
3
填写配置详情

三、开始使用

1
验证配置

重启 VS Code 并打开 Cline 面板验证配置：

1. 按 Ctrl+Shift+P 打开命令面板
2. 输入 "Cline: Start new conversation"
3. 输入测试消息："Hello, KAT-Coder!"
2
使用示例
# 代码生成
"Create a React component for a user profile"

# 代码审查
"Review this code and suggest improvements: [粘贴代码]"

# 代码优化
"Optimize this function: [粘贴代码]"

配置完成后，您可以在输入框中输入请求，模型将帮助您完成各种任务，例如：

  • 代码生成和补全
  • 调试和错误修复
  • 代码解释和文档
  • 重构和优化
提示： Cline 支持多种编程语言，包括 JavaScript、Python、Java、C++ 等。
